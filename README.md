# week7-e-commerce-backend
The Developers Arena Internship
## E-Commerce Backend – Spring Boot

### Features
- Product catalog & categories
- Orders & order items
- Payments with status tracking
- User management
- Flyway DB migrations
- PostgreSQL + JPA
- Docker support

### Tech Stack
Spring Boot, Java 17, PostgreSQL, Flyway, Docker

### Run
docker-compose up
mvn spring-boot:run
